▶ What It Does


YoAhorro IA is an intelligent price-comparison ecosystem for the Colombian retail market. Users scan any grocery or consumer product with their phone camera — Gemini Vision API identifies it in milliseconds — and the system returns a ranked list of prices across Alkosto, Éxito, Jumbo, Carulla, Falabella and MercadoLibre, scraped in real time and cached via Redis. The cheapest option always rises to the top.

▶ Engineering Highlights


Built a parallelized scraping engine that fires all store scrapers simultaneously using Promise.allSettled(), with a Redis caching layer (TTL-configurable) that eliminates redundant requests. ScrapingBee proxy handles anti-bot bypass against VTEX-based retailer APIs, achieving 98.7% proxy uptime and zero IP bans across months of continuous operation.


Integrated Google Gemini Vision API to identify products from a photo with a single REST call. The pipeline extracts product name, brand, and category from the image, then queries the DB and falls back to live scraping — all in under 35ms average. The admin dashboard provides real-time scan activity logs, store health checks (PostgreSQL, Redis, ScrapingBee, Wompi), and MRR metrics updated every 5 minutes via NestJS cron jobs.


Payment and subscription flows run through Wompi, Colombia's leading payment gateway, with a mock fallback system for degraded scraping conditions ensuring zero-downtime UX even when external retailer APIs rate-limit the service.
